# umjik

A declarative animation library for React Native with a framer-motion compatible API, powered by Reanimated 3.

## Why umjik?

React Native animations with Reanimated require too much boilerplate — `useSharedValue`, `useAnimatedStyle`, `withTiming` for even a simple fade-in. On the web, framer-motion solved this elegantly, but it's web-only.

umjik bridges this gap: **framer-motion compatible API + Reanimated 3 engine** for React Native.
